# Sharpheus Game Engine

Easy-to-use 2D C++ game engine

*Under Apache License 2.0*

### Main features:
+ 2D Rendering
+ Collision system
+ Event system
+ Editor

### Supported platforms:
+ Windows
+ *Linux (in future)*

### Supported rendering APIs:
+ OpenGL

### Using versions:
+ C++ 17
+ Premake 5.0 alpha 15
+ GLFW 3.3.2
