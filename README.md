# Sharpheus Game Engine

Easy-to-use 2D C++ game engine

*Under Apache License 2.0*

![Sharpheus](/Assets/Branding/sharpheus_promo.png?raw=true "Sharpheus")

### Main features:
+ 2D Rendering
+ Collision system
+ Event system
+ Editor

### Supported platforms:
+ Windows
+ *Linux (in future)*

### Supported rendering APIs:
+ OpenGL

### Using versions:
+ C++ 17
+ OpenGL 4.6
+ GLFW 3.3.2
+ GLEW 2.1.0
+ glm 0.9.9.8
+ spdlog 1.8.2
+ Premake 5.0 alpha 15
